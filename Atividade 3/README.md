# 📊 Análise de Vendas com Pandas

Projeto simples de análise de dados usando Python.

---

## 🚀 Funcionalidades

* 📅 Faturamento por mês
* 📈 Geração de gráfico automático
* 🏆 Melhor vendedor por ticket médio
* 💰 Análise de vendas acima de 200
* 🧹 Tratamento de valores nulos

---

## ▶️ Como rodar

```bash
pip install -r requirements.txt
python src/analise.py
```

---

## 📊 Saída

O projeto gera:

* Gráfico salvo como `grafico.png`
* Resultados no terminal

---

```
analise-vendas/
```
