# 📊 Análise de Vendas com Python

Projeto de análise exploratória de dados de vendas utilizando **Pandas** e **Matplotlib**.

---

## 🚀 Funcionalidades

* 📅 Evolução do faturamento mês a mês
* 📈 Gráfico de tendência de vendas
* 🏆 Identificação do vendedor com maior ticket médio
* 💰 Análise de vendas acima de 200
* 🧹 Tratamento de dados nulos

---

## 🛠️ Tecnologias

* Python
* Pandas
* Matplotlib
* NumPy

---

## ▶️ Como executar

```bash
pip install -r requirements.txt
python src/analise.py
```
